TurnDesk — Logo Pack
====================

The mark: technicians (dots) rotating fairly around the front-desk dashboard (the
grid at the center); the gold dot marks whose turn is next. Concept = "the fair turn."

Colors
  Teal (primary)     #1a5252
  Deep teal          #0f3d3d
  Mint (light teal)  #8fd4d3
  Gold (accent)      #f5c870
  Wordmark font      Manrope (ExtraBold for "TurnDesk")

Files
  SVG (vector — scales forever, best for print/web that accepts SVG)
    turndesk-mark.svg           the mark alone (dashboard grid center) — light backgrounds
    turndesk-mark-compact.svg   the mark with a solid dot center — best for tiny sizes / favicons
    turndesk-appicon.svg        the mark on a rounded teal tile — app icon / profile picture

  png/ (raster — use these for social media, which usually needs PNG/JPG)
    turndesk-appicon-1024.png       1024x1024 square teal icon — profile pic / avatar / app icon
    turndesk-mark-512.png           512x512 mark on transparent — overlays / watermarks
    turndesk-logo-white-1600.png    horizontal logo + wordmark on white — headers, light posts
    turndesk-logo-dark-1600.png     horizontal logo + wordmark on dark teal — dark posts

Tip: need a different size or a JPG? Open any SVG in a browser or Canva and export at
whatever dimensions you want — vectors never lose quality.
